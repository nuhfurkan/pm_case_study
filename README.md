## Set up
Run following commands (on windows)
```bash
python -m venv pm_case_study_env
.\pm_case_study_env\Scripts\activate 
pip install -r requirements.txt
```

Run following commands (on linux/mac)
```bash
python -m venv pm_case_study_env
.\pm_case_study_env/bin/activate 
pip install -r requirements.txt
```
